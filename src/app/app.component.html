Given an original query and a follow-up query, label a span within the follow-up query that represents the additional information that the follow-up query adds to the original query.

The task is to:

  1. Determine if there is a span in follow-up query that is a Label.
  2. Identify the span in follow-up query that can serve as a label.
    1. If there is no such span, identify a possible non-span label.
    2. if there is no possible label, mark the answer as “N/A”.
